Konfiguruj Git:
git config --global user.name "Twoje imię"
git config --global user.email "youremail@domain.com"

inne polecenia
git clone (url)
git add (nazwy plików)
git commit -m "(zatwierdzony komunikat)
git push -u origin master
git remote add origin master (url)
